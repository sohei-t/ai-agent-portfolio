## 使い方

### 基本操作

1. アプリを起動すると、ピアノの鍵盤が表示されます
2. 鍵盤をタップ/クリックすると音が鳴ります
3. 白鍵と黒鍵が正確に配置されています

### オクターブ切り替え

- 画面上部の「C4-C5」「C5-C6」ボタンで演奏範囲を切り替えられます
- 2オクターブ分の音域をカバーしています

### テーマ変更

画面右上のテーマアイコンから、5種類のテーマを選択できます：

- デフォルト（ブルー）
- サンセット（オレンジ）
- フォレスト（グリーン）
- ミッドナイト（ダークブルー）
- チェリー（ピンク）

### 音量調整

画面下部のスライダーで音量を0-100%の範囲で調整できます。
