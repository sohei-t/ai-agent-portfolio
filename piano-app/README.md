# Piano App - Mobile-First Piano Application

スマートフォン向けのインタラクティブなピアノ演奏Webアプリケーション。React 18 + Web Audio APIで構築。

## About

**[プロジェクト解説ページ](https://sohei-t.github.io/ai-agent-portfolio/piano-app/about.html)** - 技術詳細・開発プロセス・アーキテクチャの解説

## Features

- **18種類の楽器音源**: ピアノ、オルガン、ギター、動物の鳴き声など
- **9曲の自動演奏**: ジングルベル、エリーゼのためになど有名曲を収録
- **ドラムマシン**: 14種類のリズムパターン
- **録音機能**: 演奏を録音・再生・ダウンロード
- **5つのテーマ**: 気分に合わせて選べるカラーテーマ
- **モバイル対応**: スマホ・タブレット最適化
