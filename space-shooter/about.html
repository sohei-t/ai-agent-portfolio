Project Overview

Space Shooter is a browser-based arcade game inspired by the classic Space Invaders, rebuilt with modern web technologies and enhanced with contemporary game design principles. Featuring a cyberpunk neon aesthetic, responsive design, and advanced gameplay mechanics, this project demonstrates the power of HTML5 Canvas and vanilla JavaScript.

14 Modules
92.24% Test Coverage
615 Tests
60 Target FPS

Built entirely without frameworks, this game showcases clean architecture, comprehensive testing, and performance optimization techniques. The codebase follows software engineering best practices with modular design, extensive documentation, and maintainable code structure.

Key Features

Gameplay Innovations

Power-Up System

Three unique power-ups that can stack for strategic gameplay:

  • Rapid Fire: Double your shooting speed (300ms ‚Üí 150ms)
  • Shield: Temporary invincibility with visual effects
  • Multi-Shot: Fire 3 bullets in spread pattern

Stack up to 3 power-ups simultaneously with independent timers.

Combo System

Reward precise play with score multipliers:

  • 2x multiplier at 5 consecutive hits
  • 3x multiplier at 10+ consecutive hits
  • Resets after 2 seconds or missed shots

Master the combo system to achieve high scores!

Adaptive Performance

Automatically adjusts quality based on device capabilities:

  • 60 FPS on high-end devices
  • 45 FPS on mid-range devices
  • 30 FPS minimum on older hardware

Reduces particle effects when FPS drops below 40.

Responsive Design

Perfect gameplay on any device:

  • Fixed 4:3 aspect ratio (800√ó600 logical)
  • CSS scaling from 320px to 2560px
  • Touch controls on mobile devices
  • Landscape and portrait support

30% larger touch zones for comfortable mobile play.

Enemy AI

Classic movement with modern precision:

  • 5√ó11 grid formation (55 enemies)
  • Synchronized movement patterns
  • Speed increases 10% per wave
  • 1% random fire rate per enemy per frame

Progressive difficulty scaling for long-term engagement.

Visual Effects

Cyberpunk neon aesthetic with particle systems:

  • Explosion particles on enemy destruction
  • Power-up glow effects
  • Neon color palette (cyan/magenta/yellow)
  • Smooth animations at 60 FPS

Up to 100 particles on high-end devices.

Technical Architecture

14-Module System

The game is built on a clean, modular architecture with clear separation of concerns:

Core Modules (4)

  • Game.js: Main game loop, state machine (MENU ‚Üí PLAYING ‚Üí PAUSED ‚Üí GAME_OVER)
  • Canvas.js: Rendering engine with responsive scaling
  • InputHandler.js: Unified keyboard and touch input
  • PerformanceMonitor.js: FPS tracking and adaptive quality

Entity Modules (4)

  • Player.js: Player entity with movement, shooting, and lives
  • Enemy.js: Enemy entity and grid management (5-step movement algorithm)
  • Bullet.js: Projectile entity for both player and enemies
  • PowerUp.js: Power-up drops with 3 types

System Modules (4)

  • CollisionSystem.js: AABB collision detection (<10ms for 5,500 checks)
  • PowerUpSystem.js: Power-up state, stacking rules, and timers
  • ScoreSystem.js: Score calculation with combo multipliers
  • ParticleSystem.js: Explosion effects and visual feedback

UI Modules (3)

  • Menu.js: Start menu screen
  • HUD.js: In-game display (score, lives, power-ups, combo)
  • GameOver.js: End screen with restart option

Utility Modules (2)

  • Vector2D.js: 2D vector mathematics
  • Config.js: Centralized game configuration

// AABB Collision Detection (O(n*m) optimized)
CollisionSystem.checkCollision(rect1, rect2) {
  return !(
    rect1.right < rect2.left ||
    rect1.left > rect2.right ||
    rect1.bottom < rect2.top ||
    rect1.top > rect2.bottom
  );
}

Canvas Scaling Algorithm

// Maintain 4:3 aspect ratio while fitting viewport
const LOGICAL_WIDTH = 800;
const LOGICAL_HEIGHT = 600;

function resizeCanvas() {
  const scale = Math.min(
    window.innerWidth / LOGICAL_WIDTH,
    window.innerHeight / LOGICAL_HEIGHT
  );
  canvas.style.width = `${LOGICAL_WIDTH * scale}px`;
  canvas.style.height = `${LOGICAL_HEIGHT * scale}px`;
}
