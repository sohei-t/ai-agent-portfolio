GAME OVERVIEW

ROBO BATTLE V3 is a fast-paced 2D robot fighting game featuring real-time online multiplayer battles. Challenge players worldwide in intense mech combat!

  • Online 2-player battles via WebRTC
  • Room-based matchmaking system
  • Low-latency P2P direct connection
  • Cross-platform play (PC & Mobile)
  • Touch controls for mobile devices

ONLINE BATTLE SYSTEM

V3 introduces a revolutionary WebRTC peer-to-peer networking system. Unlike traditional client-server models, your game data travels directly to your opponent's device, ensuring minimal latency and maximum responsiveness.

How it works:
1. Create or join a room using a 6-digit code
2. Firebase handles the initial connection handshake
3. Once connected, data flows P2P (no server lag!)
4. Input sync ensures fair, responsive gameplay
