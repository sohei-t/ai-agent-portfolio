📋 概要 Overview

Picture Merge App は、2枚の写真をブラウザ上で結合し1枚の画像として出力するWebアプリです。MediaPipe Selfie Segmentation によるブラウザ内機械学習で人物を自動検出し、黒背景上に横並びで配置します。

全処理がクライアントサイドで完結するため、サーバーへの画像送信は不要。プライバシーを保ちながら、高品質な画像結合が可能です。Claude Code の AI エージェントシステムにより完全自動開発されました。

Picture Merge App is a web application that combines two photos in the browser and outputs them as a single image. It uses MediaPipe Selfie Segmentation for in-browser machine learning to automatically detect people and place them side by side on a black background.

All processing is done client-side, so no images are sent to a server. This ensures privacy while enabling high-quality image merging. Fully auto-developed by Claude Code's AI Agent system.

EXIF情報を読み取り、画像の向きを自動補正。縦長/横長を自動判定して最適なレイアウトを選択。

Reads EXIF data to auto-correct image orientation. Automatically determines portrait/landscape for optimal layout.

🧠 人物検出 (ML)Person Detection (ML)

MediaPipe Selfie Segmentation でブラウザ内ML推論。人物の重心を検出してクロップ位置を自動最適化。

In-browser ML inference via MediaPipe Selfie Segmentation. Detects person centroid for auto-optimized cropping.

⚡ Web Worker 並列処理Web Worker Processing

OffscreenCanvas + Web Workers for non-blocking rendering. UI stays responsive at all times.

プレビュー上でドラッグして各画像のクロップ位置を微調整。リアルタイムで結果を確認。

Drag on preview to fine-tune crop position for each image. See results in real-time.
