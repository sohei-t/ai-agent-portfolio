for文の基本

繰り返し処理を書いてみよう

前回のおさらい

前回、「繰り返しが必要な理由」を学びましたね。同じ処理を何度も書くのは大変で、ループを使えばたった数行で済むということでした。

今回やること

いよいよ実際にfor文を書いて、繰り返し処理を体験します。if文で学んだインデントも使いますよ。

for文の書き方

for文の基本構文
for 変数 in リスト:
    繰り返したい処理
忘れないで: コロンとインデント

if文と同じく、行末に:（コロン）を書き、次の行はスペース4つ分のインデントが必要です。

最初のfor文を書いてみよう

for fruit in ["りんご", "みかん", "バナナ"]:
    print(fruit)
りんご
みかん
バナナ

リストの中身が順番にfruitという変数に入り、print(fruit)が3回実行されます。

for文の動きを追ってみよう

リストの要素が1つずつfruitに代入されて処理が実行されます
やってみよう

名前リストを表示しよう

Google Colabで実行してみよう

動画を止めて、以下のコードをGoogle Colabに入力して実行してください。

for name in ["田中", "佐藤", "鈴木"]:
    print(name + "さん、おはようございます")
うまくいきましたか？

3人分の挨拶が表示されれば成功です。たった2行で3回分の処理ができました。

ループ変数の名前

forの後ろに書く変数名は自由につけられます。中身が分かる名前をつけましょう。

# 良い例: 中身が分かる名前
for name in ["田中", "佐藤"]:
    print(name)
# 悪い例: 何のデータか分からない
for x in ["田中", "佐藤"]:
    print(x)
命名のコツ

果物のリストならfruit、名前のリストならname、数値ならnumberのように、リストの「中の1つ」を表す名前が分かりやすいです。

f文字列と組み合わせる

第3章で学んだf文字列をfor文と組み合わせると、さらに便利です。

for name in ["田中", "佐藤", "鈴木"]:
    print(f"{name}さんの日報を確認しました")
田中さんの日報を確認しました
佐藤さんの日報を確認しました
鈴木さんの日報を確認しました

for文の中にif文を入れる

for文とif文を組み合わせることもできます。インデントが2段になります。

for score in [85, 42, 91, 67]:
    if score >= 70:
        print(f"{score}点: 合格")
    else:
        print(f"{score}点: 不合格")
インデントに注意

for文のインデント（スペース4つ）の中にif文を書くので、if文の中の処理はスペース8つになります。Google Colabは自動でインデントしてくれるので安心です。

ここまでのポイント

for文の基本まとめ

  • for 変数 in リスト: が基本の書き方
  • コロンとインデントを忘れない（if文と同じ）
  • ループ変数には中身が分かる名前をつける
  • f文字列やif文と組み合わせられる

まとめ

  • for文の構文： for 変数 in リスト: で繰り返し処理を書けます
  • ループ変数： リストの要素が1つずつ変数に入って処理されます
  • 組み合わせ： f文字列やif文と組み合わせて、より実用的な処理ができます
