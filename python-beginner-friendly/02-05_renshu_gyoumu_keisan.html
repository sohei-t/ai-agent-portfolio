Chapter 2 - 練習

変数を使った
業務計算プログラム

売上高・原価・粗利益・利益率を計算しよう

この練習で作るもの

この章で学んだ変数、計算、命名ルール、整数と小数の知識をすべて使って、月の売上データから利益率を計算するプログラムを作ります。

なぜこの計算が大切か

粗利益率は、ビジネスの健全性を測る最も基本的な指標です。「売上がいくらあっても、利益が出ているか」を数字で確認できます。

ステップ 1

売上高と原価を変数に入れよう

Google Colabで入力してください

まず、基本データを変数に入れます。

# 月次売上データ
sales = 5000000      # 売上高: 500万円
cost = 3200000       # 原価: 320万円
変数名に注目

a や b ではなく、sales（売上）と cost（原価）という分かりやすい名前をつけています。前のトピックで学んだスネークケースも活用します。

ステップ 2

粗利益を計算しよう

動画を止めて、自分で書いてみましょう

売上高から原価を引いて、粗利益を計算してください。

# 粗利益の計算
gross_profit = sales - cost
print(gross_profit)
1800000

粗利益は180万円です。売上500万から原価320万を引いた結果ですね。

ステップ 3

利益率を計算しよう

動画を止めて、自分で書いてみましょう

粗利益を売上高で割って、100を掛けるとパーセントになります。

# 利益率（パーセント）
profit_rate = gross_profit / sales * 100
print(profit_rate)
36.0

利益率は36%です。割り算を使うのでfloatの36.0と表示されます。前のトピックで学んだ通りですね。

ここまでの確認

ここまでの流れ

  • 売上高と原価を変数に入れた（分かりやすい名前）
  • 粗利益 = 売上高 - 原価 で計算した
  • 利益率 = 粗利益 / 売上高 * 100 でパーセントを求めた
  • 割り算の結果はfloat（36.0）になった

完成プログラム

ステップ1~3をまとめると、以下の完成プログラムになります。

# 月次売上レポート
sales = 5000000
cost = 3200000

gross_profit = sales - cost
profit_rate = gross_profit / sales * 100

print(sales)
print(cost)
print(gross_profit)
print(profit_rate)
5000000
3200000
1800000
36.0
たった8行で業務計算ができました

変数に名前をつけて計算するだけで、売上レポートの基本が作れます。数字を変えれば毎月使えるプログラムです。

ボーナスチャレンジ

経費を追加してみよう

余裕がある方は挑戦してみましょう

粗利益から経費を引いて「営業利益」を計算するコードを追加します。

# 経費を追加
sales = 5000000
cost = 3200000
expenses = 800000    # 経費: 80万円

gross_profit = sales - cost
operating_profit = gross_profit - expenses
op_rate = operating_profit / sales * 100

print(gross_profit)
print(operating_profit)
print(op_rate)
1800000
1000000
20.0

第2章のまとめ

  • 変数：データに名前をつけて保存する「名前付きの箱」

  • 数値の型：int（整数）とfloat（小数）の2種類
  • 次の章：文字列の扱い方を学びます
