今回のミッション

見積書の計算ロジックを関数で整理しましょう！

シナリオ
あなたは営業部のアシスタントです。お客様への見積書を作成するため、
税込計算、割引計算、合計計算を行うプログラムを作ります。
関数を使って、再利用しやすいコードにしましょう！
使う技術
def文（関数の定義）、引数（データを渡す）、return（結果を返す）、
デフォルト引数、for文（リストの繰り返し）
ステップ 1

税込計算関数を作ろう

動画を止めて、Google Colabで作りましょう

価格を受け取って、税込価格を返す関数を作ってください。

def calc_tax(price, rate=1.1):
    return int(price * rate)

# テスト
print(calc_tax(1000))
print(calc_tax(500, 1.08))
1100
540
ポイント
デフォルト税率は10%。食品など8%の場合は第2引数で指定できます。
ステップ 2

割引計算関数を作ろう

動画を止めて、続きを作りましょう

割引率を受け取って、割引後の価格を返す関数を追加してください。

def discount(price, rate=0.1):
    return int(price * (1 - rate))

# テスト
print(discount(3000))
print(discount(3000, 0.2))
2700
2400
ステップ 3

合計計算関数を作ろう

リストの合計を計算する関数です

価格のリストを受け取って、合計を返す関数を作りましょう。

def calc_total(prices):
    total = 0
    for p in prices:
        total = total + p
    return total

# テスト
print(calc_total([1100, 2200, 550]))
3850
for文を復習！
第5章で学んだfor文を使って、リストの各要素を合計しています。
ステップ 4

関数を組み合わせよう

動画を止めて、全部を組み合わせましょう

3つの関数を使って、見積書の計算をしてみましょう。

# 商品リスト（税抜価格）
items = [1500, 3000, 800]

# 各商品の税込価格を計算
tax_prices = []
for item in items:
    tax_prices.append(calc_tax(item))

# 合計を計算
total = calc_total(tax_prices)
print("税込合計: " + str(total) + "円")
税込合計: 5830円
ステップ 5（最終）

見積書を完成させよう

全関数を使った見積書プログラム

税込 → 割引 → 合計の流れで見積書を作ります。

items = [1500, 3000, 800]

print("=== 見積書 ===")
tax_prices = []
for item in items:
    tp = calc_tax(item)
    tax_prices.append(tp)
    print(str(item) + "円 → 税込 " + str(tp) + "円")

total = calc_total(tax_prices)
final = discount(total, 0.05)
print("----------")
print("小計: " + str(total) + "円")
print("5%割引後: " + str(final) + "円")
=== 見積書 ===
1500円 → 税込 1650円
3000円 → 税込 3300円
800円 → 税込 880円
----------
小計: 5830円
5%割引後: 5538円

第7章で学んだこと

  • 関数は「入力 → 処理 → 出力」（自動販売機のたとえ）
  • def文で関数を定義し、関数名()で呼び出す
  • 引数でデータを渡し、デフォルト引数で省略可能に
  • returnで結果を返し、変数に保存して再利用
  • 複数の関数を組み合わせて複雑な処理を実現

第7章のまとめ

  • 関数の3要素 def（定義）、引数（入力）、return（出力）
  • 再利用の力 一度作った関数は何度でも使える。修正も1か所でOK
  • 組み合わせ 関数同士を組み合わせて、複雑な業務ロジックを構築できる
  • 次の章 エラーへの対処やライブラリの紹介、そして総まとめです！
