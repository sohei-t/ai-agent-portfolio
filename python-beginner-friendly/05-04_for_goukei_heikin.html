for文で計算しよう

合計と平均 - データ集計の基本

前回のおさらい

前回、range関数を使って「N回繰り返す」方法を学びましたね。for文とrangeを組み合わせて、指定した回数だけ繰り返し処理ができるようになりました。

今回の目標

for文を使ってデータの合計と平均を計算します。ExcelのSUM関数と同じことをプログラムで実現してみましょう。

合計を計算する基本パターン

for文で合計を計算するには、「0から始めて足していく」というパターンを使います。

total = 0  # まず0からスタート
for value in [100, 200, 300]:
    total = total + value
print(f"合計: {total}円")
合計: 600円
ExcelのSUM関数と同じ

Excelで=SUM(A1:A3)と書くのと同じ計算です。プログラムでは「0から始めて1つずつ足す」という手順で実現します。

合計計算の動きを見てみよう

100 total = 100 + 200 → total は 300 total = 300 + 300 → total は 600 ループ終了 → 合計: 600円
totalが0からスタートし、ループのたびに値が加算されていきます

短く書ける += の記号

total = total + value は total += value と短く書けます。

total = 0
for value in [100, 200, 300]:
    total += value  # total = total + value と同じ
print(f"合計: {total}円")
どちらで書いてもOK

total = total + value でも total += value でも結果は同じです。慣れてきたら短い方を使ってみましょう。

やってみよう

5日分の売上合計を計算しよう

Google Colabで実行してみよう

動画を止めて、以下のコードを入力して実行してください。

sales = [12000, 8500, 15000, 9200, 11300]

「5日分の売上合計: 56000円」と表示されれば成功です。

平均を計算しよう

平均は「合計 ÷ 個数」で求められます。個数はlen()関数で数えられます。

sales = [12000, 8500, 15000, 9200, 11300]
total = 0
for s in sales:
    total += s
average = total / len(sales)
print(f"平均売上: {average}円")
平均売上: 11200.0円
len()関数を思い出そう

第3章で文字列の長さを数えるのに使ったlen()関数は、リストの要素数を数えるのにも使えます。len(sales)は5を返します。

業務例: 売上レポート

sales = [12000, 8500, 15000, 9200, 11300]
total = 0
for s in sales:
    total += s
average = total / len(sales)
print(f"合計: {total}円")
print(f"平均: {average}円")
ExcelのAVERAGE関数と同じ

ExcelでAVERAGE(A1:A5)と書くのと同じ結果です。「合計÷個数」が平均の基本公式です。

おまけ: 最大値を見つける

for文とif文を組み合わせて、最大値を見つけることもできます。

sales = [12000, 8500, 15000, 9200]
max_sale = sales[0]  # 最初の値を仮の最大値に
for s in sales:
    if s > max_sale:
        max_sale = s
print(f"最高売上: {max_sale}円")
少し難しいですが大丈夫

ここまでのポイント

合計と平均のまとめ

  • 合計はtotal = 0からスタートしてfor文で足していく
  • +=は「加算して代入」の短縮記法
  • 平均は「合計 ÷ len(リスト)」で計算する
  • ExcelのSUM・AVERAGE関数と同じ計算ができる

まとめ

  • 合計のパターン： total = 0 → for文で total += 値 と加算
  • 平均のパターン： 合計 ÷ len(リスト) で個数を割る
  • Excel対応： SUM関数 = for文の合計、AVERAGE関数 = 合計÷個数
