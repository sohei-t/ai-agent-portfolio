勤怠管理プログラムを作ろう

全8章の知識を使った総合ミニプロジェクト

作るもの: 勤怠管理プログラム

この総合練習では、研修で学んだ全ての技術を使って、実践的な勤怠管理プログラムを作ります。

Step 1: 社員データを作る

まず、社員の情報を辞書のリストで用意します。第6章で学んだリストと辞書の組み合わせです。

employees = [
    {"name": "田中", "dept": "営業部"},
    {"name": "佐藤", "dept": "開発部"},
    {"name": "鈴木", "dept": "総務部"},
]
ポイント
リストの中に辞書を入れると、「複数の人」の「複数の情報」を管理できます。Excelの表のようなイメージです。

Step 2: 勤怠データを作る

次に、各社員の今月の勤務時間をリストで用意します。

work_hours = {
    "田中": [8, 9, 8, 7, 10],
    "佐藤": [8, 8, 8, 8, 8],
    "鈴木": [7, 6, 9, 8, 7],
}

辞書のキーが社員名、値が5日分の勤務時間リストです。辞書+リストの組み合わせで、複雑なデータも管理できます。

コツ
カッコと波括弧の閉じ忘れに注意してください。前のトピックで学んだSyntaxErrorが出たら、対になる記号を確認しましょう。

Step 3: 集計関数を作る

第7章で学んだ関数を使って、合計と平均を計算する関数を作ります。

def calc_total(hours):
    total = 0
    for h in hours:
        total = total + h
    return total
def calc_average(hours):
    total = calc_total(hours)
    return total / len(hours)
注目ポイント
calc_averageの中でcalc_totalを呼び出しています。関数の中から別の関数を呼べる、これが「再利用」の力です。

Step 3（続き）: 勤務状況を判定する

if文を使って、平均勤務時間から勤務状況を判定する関数を作ります。

def check_status(avg):
    if avg >= 9:
        return "残業多め - 要注意"
    elif avg >= 7:
        return "適正"
    else:
        return "勤務不足 - 要確認"

第4章で学んだif-elif-elseを使って、3段階の判定をしています。

やってみよう

関数を入力しよう

動画を止めて、3つの関数（calc_total、calc_average、check_status）をGoogle Colabに入力してください。入力したら、テストしてみましょう。

# テスト: 田中さんのデータで試す
hours = work_hours["田中"]
print(calc_total(hours))
print(calc_average(hours))
期待される結果
田中さんの合計: 42時間、平均: 8.4時間と表示されれば正解です。

Step 4: レポートを出力する

全社員の勤怠データをfor文で処理し、f文字列でレポートを出力します。datetimeで日付も自動取得します。

import datetime
today = datetime.date.today()
print(f"=== 勤怠レポート {today} ===")
for emp in employees:
    name = emp["name"]
    hours = work_hours[name]
    avg = calc_average(hours)
    status = check_status(avg)
    print(f"{name}({emp['dept']}): 平均{avg:.1f}h - {status}")
