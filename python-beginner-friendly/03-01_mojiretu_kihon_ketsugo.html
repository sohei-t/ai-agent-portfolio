文字列の作り方と結合

文字をプログラムで扱う第一歩。クォーテーションと+を覚えよう

こんな場面、ありませんか？

Excelでセルに文字を入力するように、Pythonでも文字を扱えます。
今日はその基本中の基本を学びましょう。

文字列とは？

Pythonで文字を扱うときは、クォーテーションで囲みます。
囲まれた文字のかたまりを文字列と呼びます。

" こんにちは "
Excelで言うと
セルに直接文字を入力するのと似ています。ただしPythonでは、文字をクォーテーションで囲むルールがあります。

シングルとダブル、どちらでもOK

# どちらも同じ結果になります
print("こんにちは")
print('こんにちは')

こんにちは
どちらを使うべき？
どちらでもOKですが、チームで統一するのがおすすめです。この研修では"ダブルクォーテーション"を使います。

文字列を表示してみよう

Google Colabで試してみましょう

次のコードを入力して実行してください。

print("私の名前はPythonです")

好きな文字に変えても試してみてください。

+で文字列を結合する

文字列どうしは + で繋げることができます。

greeting = "こんにちは、" + "田中さん！"
print(greeting)
こんにちは、田中さん！
文字列の + は「つなげる」という意味

変数を使って結合する

第2章で学んだ変数に文字列を入れて、+で繋げることもできます。

name = "田中"
print("おはようございます、" + name + "さん！")
おはようございます、田中さん！
ポイント
変数nameの中身が「田中」なので、それが文字列の間に挟み込まれます。名前を変えれば結果も変わります。
やってみよう

名前と挨拶を組み合わせよう

コードを入力して実行してみましょう
name = "自分の名前"
print("ようこそ、" + name + "さん！")

「自分の名前」の部分をあなたの名前に変えて実行してください。上司や同僚の名前にしても面白いですよ。

数字 + 文字列 = エラー！

ここで一つ、大事な落とし穴を体験しましょう。

age = 25
print("年齢は" + age + "歳です")
TypeError: can only concatenate str (not "int") to str
なぜエラー？
Pythonでは文字列と数値は別の種類のデータです。「文字列」と「数値」を + で繋げることはできません。Excelとの大きな違いです。

str()で数値を文字列に変換

str()を使えば、数値を文字列に変換できます。

age = 25
print("年齢は" + str(age) + "歳です")
年齢は25歳です
覚えておこう
型変換についてはこの章の後のトピックで詳しく学びます。今は「str()で数字を文字にできる」とだけ覚えておけばOKです。
やってみよう

エラーを体験してみよう

まずエラーを出して、次に直してみましょう

ステップ1: わざとエラーを出す

age = 25
print("年齢は" + age + "歳です")

ステップ2: str()を使って直す

age = 25
print("年齢は" + str(age) + "歳です")
ここまでのポイント

第3章 トピック1 チェックポイント

  • 文字列はクォーテーションで囲む（シングルでもダブルでもOK）
  • 文字列どうしは + で結合できる
  • 数値と文字列を + で繋げるとTypeError
  • str()で数値を文字列に変換すれば解決

まとめ

  • 文字列の作り方 : ダブルクォーテーション " " またはシングルクォーテーション ' ' で囲む
  • 文字列の結合 : + を使って複数の文字列を1つに繋げる
  • 型に注意 : 数値と文字列は + で繋げられない。str()で変換が必要
