関数にデータを渡す

自動販売機に「何がほしいか」を伝える方法

毎回同じ挨拶では困る

前回作ったgreet関数は、いつも「こんにちは！」しか言えません。

def greet():
    print("こんにちは！")

greet()  # いつも同じ...
もっと柔軟にしたい！
「田中さん、こんにちは！」「鈴木さん、こんにちは！」のように、
相手によって挨拶を変えたいですよね。そこで「引数」の出番です！

引数 = お金を入れること

引数を使うと、同じ関数で異なる結果が得られます

引数の書き方

カッコの中に「受け取るデータの名前」を書きます

def greet(name):
    print(name + "さん、こんにちは！")
greet("田中")
greet("鈴木")
田中さん、こんにちは！
鈴木さん、こんにちは！
ポイント
defのカッコ内に書いた name が引数です。
呼び出すとき渡した「田中」が、nameに入って処理されます。

引数付き関数を作ろう

動画を止めて、Google Colabで試しましょう

名前を受け取って挨拶する関数を作ってみましょう。

def greet(name):
    print(name + "さん、こんにちは！")

greet("田中")
greet("鈴木")
greet("佐藤")
できましたか？
同じ関数なのに、渡す名前を変えるだけで結果が変わりますね。これが引数の力です！

複数の引数を渡す

カンマで区切って、複数のデータを渡せます

def calc_tax(price, tax_rate):
    result = price * tax_rate
    print(result)
calc_tax(1000, 1.1)
calc_tax(2500, 1.1)
1100.0
2750.0
業務での活用
税込計算を関数にすれば、商品の価格と税率を渡すだけで計算完了！
Excelの関数に数値を入れるのと同じ感覚です。

引数の順番に注意

引数は定義した順番と同じ順番で渡す必要があります。

def show_info(name, department):
    print(name + " / " + department)
show_info("田中", "営業部")  # 正しい
show_info("営業部", "田中")  # 逆になる！
田中 / 営業部
営業部 / 田中
順番を間違えると...
エラーにはなりませんが、結果がおかしくなります。
定義と呼び出しの順番を合わせましょう！

デフォルト引数

引数に「初期値」を設定できます。省略されたときに使われます。

def calc_tax(price, tax_rate=1.1):
    result = price * tax_rate
    print(result)
calc_tax(1000)        # 税率省略 → 1.1
calc_tax(1000, 1.08)  # 税率指定 → 1.08
1100.0
1080.0
便利！
「普通は10%だけど、たまに8%」のような場合にピッタリ。
よく使う値をデフォルトにしておけば、毎回書く手間が省けます。
やってみよう

税込計算関数を作ろう

動画を止めて、Google Colabで試しましょう

デフォルト引数付きの税込計算関数を作ってみましょう。

def calc_tax(price, tax_rate=1.1):
    result = price * tax_rate
    print(int(result))

calc_tax(980)
calc_tax(500, 1.08)
1078
540
すばらしい！
税率を省略すれば10%、指定すれば軽減税率の8%。

社員紹介関数を作ろう

チャレンジ問題です

名前、部署、役職を受け取って表示する関数を作りましょう。

def introduce(name, dept, role="メンバー"):
    print(dept + "の" + name + "です。")
    print("役職: " + role)

introduce("田中", "営業部", "課長")
introduce("鈴木", "開発部")
営業部の田中です。
役職: 課長
開発部の鈴木です。
役職: メンバー
ここまでのポイント

引数の基本

  • 引数はカッコの中に書く。関数にデータを渡す仕組み
  • カンマで区切れば複数の引数を渡せる
  • 引数の順番は定義と呼び出しで合わせる
  • デフォルト引数で「よく使う値」を初期設定できる

まとめ

  • 引数 関数にデータを渡す仕組み。自動販売機に入れるお金と同じ
  • 複数引数 カンマ区切りで何個でも渡せる。順番に注意
  • デフォルト引数 省略時の初期値を設定できる（tax_rate=1.1など）
