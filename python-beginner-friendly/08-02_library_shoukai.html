便利な道具箱
ライブラリ紹介

誰かが作ってくれた便利な関数の集まりを使ってみよう

ライブラリとは？

第7章で、自分で関数を作る方法を学びましたね。ライブラリとは、誰かが作ってくれた便利な関数をまとめた「道具箱」のようなものです。

Excelで言うと
Excelの関数（SUM、VLOOKUP、TODAYなど）は最初から使えますよね。Pythonのライブラリも同じで、importするだけですぐ使えます。

import文の書き方

ライブラリを使うには、最初にimportと書きます。「このライブラリを使いますよ」という宣言です。

import datetime

たったこの1行で、日付や時刻に関する便利な関数が全て使えるようになります。importは、プログラムの一番最初に書くのがルールです。

覚え方
import = インポート = 「取り込む」という意味です。道具箱から必要な道具を取り出すイメージです。

datetimeで今日の日付を取得

datetimeライブラリを使うと、今日の日付や現在の時刻を取得できます。

import datetime

today = datetime.date.today()
print(today)
2026-02-19
業務での活用例
今日の日付を自動で取得して、議事録のヘッダーに入れたり、レポートの日付を自動設定したりできます。手入力のミスがなくなります。
やってみよう

今日の日付を表示しよう

動画を止めて、Google Colabで以下のコードを実行してみましょう。今日の日付が表示されるはずです。

import datetime

today = datetime.date.today()
print(f"今日の日付: {today}")

できたら、f文字列を使って「議事録 - 2026年2月19日」のような形式で表示してみましょう。

ヒント
today.yearで年、today.monthで月、today.dayで日を取得できます。

実践: 議事録ヘッダーの自動生成

f文字列とdatetimeを組み合わせると、議事録のヘッダーを自動生成できます。

import datetime

today = datetime.date.today()
header = f"議事録 - {today.year}年{today.month}月{today.day}日"
print(header)
議事録 - 2026年2月19日

日付を手で入力する必要がなくなりました。これだけでも業務の小さな効率化になりますね。

他にもこんなライブラリが

Pythonには、最初からたくさんのライブラリが入っています。今日は2つだけ紹介します。

math
math（数学）
平方根、円周率、四捨五入など、数学的な計算ができます。math.sqrt(16)で4.0が返ります。
random
random（ランダム）
ランダムな数を作れます。くじ引きや、サンプルデータの生成に便利です。
すごいですね
importの1行を書くだけで、世界中のプログラマーが作った便利な道具を使えるようになります。これがPythonの大きな魅力です。

mathライブラリを試そう

動画を止めて、以下のコードをGoogle Colabで実行してみましょう。

import math

print(math.sqrt(16))    # 平方根
print(math.pi)            # 円周率
print(math.ceil(3.2))    # 切り上げ
余裕がある方は
import random して random.randint(1, 100) を試してみましょう。1から100までのランダムな数が表示されます。
