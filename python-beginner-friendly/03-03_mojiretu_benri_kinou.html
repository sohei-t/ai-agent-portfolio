len() - 文字数を数える

message = "おはようございます"
print(len(message))
9
Excelで言うと
ExcelのLEN関数とまったく同じです！ =LEN("おはようございます") と同じ結果になります。

replace() - 文字を置換する

text = "今日は晴れです"
new_text = text.replace("晴れ", "雨")
print(new_text)
今日は雨です
Excelで言うと
ExcelのSUBSTITUTE関数と同じです。元の文字列は変わらず、新しい文字列が作られます。
やってみよう

len()とreplace()を試そう

2つのコードを順番に実行してみましょう
# 文字数を数える
company = "株式会社サンプル"
print(f"会社名は{len(company)}文字です")
# 文字を置換する
msg = "お疲れ様です、田中さん"
print(msg.replace("田中", "佐藤"))

split() - 文字列を分割する

data = "田中,佐藤,鈴木,高橋"
names = data.split(",")
print(names)
['田中', '佐藤', '鈴木', '高橋']
リストが返ってくる
splitの結果は「リスト」という形式です。リストについては第6章で詳しく学びます。今は「カンマで区切って分解できる」と覚えておきましょう。

業務で使える！メールのドメイン取得

email = "tanaka@example.com"
parts = email.split("@")
print(f"ユーザー名: {parts[0]}")
print(f"ドメイン: {parts[1]}")
ユーザー名: tanaka
ドメイン: example.com
業務活用ヒント
メールアドレスを@で分割すれば、ユーザー名とドメインを簡単に取り出せます。顧客管理やデータ整理で役立ちます。

upper()とlower() - 大文字・小文字変換

text = "Hello Python"
print(text.upper())
print(text.lower())
HELLO PYTHON
hello python
使い道
ユーザーの入力を比較するとき、大文字小文字を統一してから比較すると便利です。例えば「Yes」も「yes」も「YES」も同じ意味として扱えます。
やってみよう

splitとupper/lowerを試そう

メールアドレスを分解してみましょう
email = "Yamada@Company.CO.JP"
print(f"小文字: {email.lower()}")
parts = email.split("@")
print(f"名前: {parts[0]}")
print(f"ドメイン: {parts[1]}")

自分のメールアドレスに変えても試してみてください。

ドット記法のルール

replace, split, upper, lower はすべて文字列の後にドットをつけて呼び出します。

lenだけ特別
len()は len(text) のように関数として呼び出します。replace, split, upper, lower は text.replace() のようにドットをつけて呼び出します。この違いは今は深く考えず、使い方を覚えれば大丈夫です。
ここまでのポイント

文字列の便利機能チェックポイント

  • len(text) で文字数を数える
  • text.replace(A, B) でAをBに置換
  • text.split(",") で区切り文字で分割
  • text.upper() / text.lower() で大文字・小文字変換

まとめ

  • len() : 文字数を数える。ExcelのLEN関数と同じ
  • replace() : 文字を置換する。元の文字列は変わらない
  • split() : 指定した文字で分割する。メール処理などで便利
  • upper()/lower() : 大文字・小文字の変換
