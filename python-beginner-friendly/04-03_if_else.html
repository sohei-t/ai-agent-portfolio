Chapter 4

if-else
「そうでなければ」

条件がFalseの場合にも対応する2択プログラム

if文だけでは足りない場面

前回のif文は、条件がTrueのときだけ処理を実行しました。
でも、Falseの場合にも別の処理をしたいことがありませんか？

業務での例

テストの点数が70点以上なら「合格」、そうでなければ「不合格」と表示したい。
在庫があれば「出荷可能」、なければ「在庫切れ」と表示したい。

こんなときに使うのが else です。

if-else のフローチャート

Trueでもfalseでも、必ずどちらかの処理が実行されます

if-else の書き方

if 条件:
    条件がTrueのときの処理
else:
    条件がFalseのときの処理
elseのルール

1. else はインデントなしで、ifと同じ位置に書く
2. else の後にもコロン : を書く
3. elseの中身もインデント（半角スペース4つ）で書く

合格 / 不合格の判定

score = 65

if score >= 70:
    print("合格です！")
else:
    print("不合格です")
不合格です

score が 65 なので、70以上ではない。つまり条件はFalse。
だからelseの中の処理が実行されます。

在庫チェックプログラム

在庫があれば「出荷可能」、なければ「在庫切れ」と表示しましょう。

zaiko = 0

if zaiko > 0:
    print("出荷可能です")
else:
    print("在庫切れです")

zaiko を 10 に変えたら、結果はどう変わりますか？試してみてください。

業務で使える例：残業チェック

jitsu_jikan = 9.5
kitei_jikan = 8.0

if jitsu_jikan > kitei_jikan:
    print("残業があります")
    zangyou = jitsu_jikan - kitei_jikan
    print(f"残業時間: {zangyou}時間")
else:
    print("定時退社です")
残業があります
残業時間: 1.5時間
ポイント

if文の中でも変数の計算やf文字列が使えます。これまで学んだ技術は全て組み合わせられます。

Excelとの比較

ExcelのIF関数と比べてみましょう。

Excelの場合

=IF(A1>=70, "合格", "不合格")

Pythonの場合

if score >= 70:
    print("合格")
else:
    print("不合格")

やっていることは同じです。Pythonでは複数行に分けて書くので、複雑な処理も読みやすく書けます。

ここまでのポイント

if-else チェック

  • else は条件がFalseのときに実行される
  • else はifと同じ位置（インデントなし）で書く
  • else の後にもコロン : を付ける
  • if-else で「必ずどちらかが実行される」2択判定ができる
やってみよう

予算チェックプログラム

予算内か判定してみよう

申請額が予算以内なら「承認」、超えていたら「予算超過」と表示しましょう。

shinsei = 35000
yosan = 30000

if shinsei <= yosan:
    print("承認します")
else:
    chouka = shinsei - yosan
    print(f"予算超過: {chouka}円")
素晴らしい進歩です！

2択の判定ができるようになりました。次は3つ以上の選択肢があるelifを学びます。

まとめ：if-else

  • else を使うと、条件がFalseの場合の処理も書ける
  • if-else は「必ずどちらかが実行される」2択構造
  • ExcelのIF関数と同じ考え方。Pythonなら複雑な処理も書きやすい
