range関数

N回繰り返す便利な方法

前回のおさらい

前回、for文の基本を学びました。for fruit in ["りんご", "みかん"]: のようにリストの要素を順に処理できましたね。

今回の目標

リストを用意しなくても「5回繰り返す」「1から12まで」のように、回数を指定して繰り返す方法を学びます。

range(5) で5回繰り返す

for i in range(5):
    print(i)
0
1
2
3
4
注意: 0から始まる

range(5)は0, 1, 2, 3, 4の5個の数を生成します。1からではなく0から始まり、5の手前で止まります。

range(5) の中身を見てみよう

range(5)は「0から4まで」の5個の数を生成します

range(1, 6) で1から5まで

開始の数を指定したい場合はrange(開始, 終了)と書きます。

for i in range(1, 6):
    print(i)
1
2
3
4
5

range(1, 6)は「1から始めて6の手前まで」なので、1, 2, 3, 4, 5が生成されます。

rangeの使い分け

書き方 生成される数 使いどころ
range(5) 0, 1, 2, 3, 4 とにかく5回繰り返したい時
range(1, 6) 1, 2, 3, 4, 5 1から5の番号が欲しい時
range(1, 13) 1, 2, 3, ..., 12 1月から12月を処理する時
覚え方のコツ

rangeの終了の数は「その数自体は含まない」と覚えましょう。1から12が欲しければrange(1, 13)です。

月次処理の表示

Google Colabで実行してみよう

動画を止めて、以下のコードを実行してください。1月から12月までの処理完了メッセージが表示されます。

業務での活用

月次レポートや年間データの処理など、「1月から12月まで」のような繰り返しは業務でよくあるパターンです。

回数だけ繰り返したい場合

数値自体は使わず、ただ「N回繰り返す」だけの場合もあります。

for i in range(3):
    print("がんばれ！")
がんばれ！
がんばれ！
がんばれ！

この場合、変数iは使わなくても問題ありません。「3回繰り返す」という意味だけで使っています。

ここまでのポイント

range関数のまとめ

  • range(N)で0からN-1までN回繰り返す
  • range(開始, 終了)で開始から終了の手前まで
  • 終了の数自体は含まれない
  • 月次処理など業務で活用場面が多い

まとめ

  • range(N)： 0からN-1までのN個の数を生成して繰り返します
  • range(開始, 終了)： 開始から終了の手前までの数を生成します
  • 業務活用： 「1月から12月」のような月次処理に便利です
