# Python入門（初心者特化版・Google Colab対応）

プログラミング完全未経験者のための Python入門カリキュラム。Google Colabを使って、インストール不要で今すぐ始められます。業務で使える実践的な例題を通じて、Pythonの基礎を丁寧に学びます。

## 特徴

- **全9章・45トピック** の体系的なカリキュラム
- **1トピック = 1〜2概念** で詰め込みすぎない設計
- 各トピック **7〜10分の音声解説付き**
- **Google Colab対応** でインストール不要、すぐに始められる
- **業務例題中心**（消費税計算、日報テンプレート、経費精算、勤怠管理など）
- 各トピック **2回以上の「やってみよう」タイム** で手を動かして学べる

## 対象者

- プログラミング完全未経験者
- Excelは使えるがコードは書いたことがない社内エンジニア

## 学習時間の目安

**10〜12時間**（全45トピック）

## 使い方

1. `content/index.html` をブラウザで開く
2. 第0章から順番に学習を進める
3. 各トピックはスライド形式で、**矢印キー**で進める
4. **音声解説（MP3）を聴きながら**学習するのがおすすめ
5. 「やってみよう」では、動画を止めて **Google Colabで実際にコードを書く**
