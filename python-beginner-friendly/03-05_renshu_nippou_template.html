使うスキル
input() f文字列 int() 文字列結合 -- 第3章の総まとめです！

ステップ1: 情報を集める

まず、input関数で必要な情報を集めましょう。

# 日報に必要な情報を入力
date = input("日付を入力: ")
name = input("氏名を入力: ")
dept = input("部署を入力: ")
ポイント
日付・氏名・部署は文字列のままでOKなので、int()で変換する必要はありません。input関数でそのまま受け取ります。

ステップ2: 業務内容と作業時間

# 業務の詳細を入力
work = input("本日の業務内容: ")
hours = int(input("作業時間(時間): "))
tomorrow = input("明日の予定: ")
hoursだけint()で変換
作業時間は数値として扱いたい（後で計算に使える）ので、int()で変換します。業務内容と明日の予定は文字列のままでOKです。
やってみよう

ステップ1と2を入力してみよう

Google Colabに入力して実行してみましょう
date = input("日付を入力: ")

各項目に自分の情報を入力してください。まだ表示はされませんが、変数にデータが入ります。

ステップ3: f文字列でテンプレート表示

# 日報テンプレートを表示
print("========================")
print(f"日報 - {date}")
print("========================")
print(f"氏名: {name}")
print(f"部署: {dept}")
print("---")
print(f"業務内容: {work}")
print(f"作業時間: {hours}時間")
print(f"明日の予定: {tomorrow}")
print("========================")

チャレンジ：残業時間を計算しよう

作業時間が8時間を超えたら残業時間を表示

hoursが8より大きければ残業時間を計算して表示を追加してみましょう。まだif文は学んでいないので、今はシンプルに残業時間を計算するだけでOKです。

# 完成版の最後に追加
overtime = hours - 8
print(f"残業時間: {overtime}時間")

作業時間を10時間にして、残業2時間と表示されることを確認してください。

第3章で学んだこと

  • 文字列はクォーテーションで囲む、+で結合できる
  • f文字列で変数や計算式を文字列に埋め込める
  • len, replace, split, upper, lower で文字列操作
  • str() / int() / float() で型変換
  • input()でキーボードから入力を受け取れる

第3章まとめ

  • 文字列の基本 : クォーテーションで囲み、+やf文字列で組み合わせる
  • 便利な機能 : len, replace, split, upper, lower で文字列を自在に操作
  • 対話的プログラム : input関数で入力を受け取り、業務テンプレートを自動生成できる
