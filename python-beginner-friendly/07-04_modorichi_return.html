関数から結果を受け取る

自動販売機から飲み物を受け取ろう

printだけでは足りない

前回の関数は結果をprintで「表示」するだけでした。

def calc_tax(price):
    print(price * 1.1)

calc_tax(1000)  # 1100.0と表示される
困った場面
表示はできたけど、この計算結果を変数に保存して後で使いたい！
合計金額を計算したり、if文で判定したりしたい。
printは「見せるだけ」。後で使えません。

戻り値 = 出てくる飲み物

printは「見せる」だけ。returnは「渡す」ことができます

returnの書き方

printの代わりに return を使います

def calc_tax(price):
    return price * 1.1
result = calc_tax(1000)
print(result)
1100.0
ポイント
return で返した値は、変数に保存できます。
result = calc_tax(1000) で、resultに1100.0が入ります。
やってみよう

returnを使ってみよう

動画を止めて、Google Colabで試しましょう

returnで結果を返す関数を作って、変数に保存してみましょう。

def calc_tax(price):
    return price * 1.1

result = calc_tax(1000)
print("税込: " + str(int(result)) + "円")
税込: 1100円
できましたか？
returnで返した値を変数に入れて、後で自由に使えるようになりました！

returnで何が嬉しいの？

結果を変数に保存できるから、後の計算に使えます！

def calc_tax(price):
    return price * 1.1

a = calc_tax(1000)
b = calc_tax(2000)
total = a + b
print("合計: " + str(int(total)) + "円")
合計: 3300円
業務での活用
個別の税込価格を計算して、合計を出す。
printでは「見るだけ」でしたが、returnなら「計算に使える」！

print と return の違い

使い分けのコツ
「計算結果を後で使いたい」なら return。
「ただ表示したいだけ」なら print。
迷ったら return にしておくのがおすすめです！

関数の完成形

引数で受け取り、returnで返す。これが関数の完全な形です！

def calc_tax(price, rate=1.1):
    return int(price * rate)

# 使い方
item1 = calc_tax(980)
item2 = calc_tax(500, 1.08)
print("合計: " + str(item1 + item2) + "円")
合計: 1618円
おめでとうございます！
引数で入力、returnで出力。自動販売機の完成形です！
def + 引数 + return の3つがそろえば、どんな関数でも作れます。
やってみよう

割引計算関数を作ろう

動画を止めて、Google Colabで試しましょう

割引率を受け取って、割引後の価格を返す関数を作りましょう。

def discount(price, rate=0.1):
    return int(price * (1 - rate))

sale_price = discount(3000)
vip_price = discount(3000, 0.2)
print("通常割引: " + str(sale_price) + "円")
print("VIP割引: " + str(vip_price) + "円")
通常割引: 2700円
VIP割引: 2400円
やってみよう

合計計算に挑戦

returnの値を組み合わせてみましょう

複数の関数呼び出しの結果を合計してみましょう。

def calc_tax(price, rate=1.1):
    return int(price * rate)

items = [500, 1200, 800]
total = 0
for item in items:
    total = total + calc_tax(item)
print("合計: " + str(total) + "円")
合計: 2750円
素晴らしい！
for文と関数の組み合わせ！第5章のループと第7章の関数が合体しました。
