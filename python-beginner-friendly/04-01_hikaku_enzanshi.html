6つの比較演算子

演算子意味使用例結果
==等しい5 == 5True
!=等しくない5 != 3True
<より小さい3 < 5True
>より大きい3 > 5False
<=以下5 <= 5True
>=以上3 >= 5False

要注意：== と = は違う！

初心者がもっともよく間違えるポイントです

よくある間違い

True と False - ブール値

比較の結果は、必ず True か False のどちらかです。

print(10 > 5)    # True
print(10 == 5)   # False
print(3 != 3)    # False
bool型（ブール型）

True と False は Python では bool型（ブール型）と呼ばれるデータ型です。
第2章で学んだ int型 や float型 と同じように、型の1つです。

変数を使って比較する

数値を直接書くだけでなく、変数同士でも比較できます。

uriage = 850000
mokuhyou = 1000000
print(uriage >= mokuhyou)
False

売上が目標以上かどうかを、変数で分かりやすく書けました。
これが業務プログラムの基本パターンです。

文字列も比較できる

数値だけでなく、文字列の比較もできます。

status = "完了"
print(status == "完了")   # True
print(status != "未着手") # True
文字列比較のポイント

大文字と小文字、全角と半角は別の文字として扱われます。
"OK" == "ok" は False です。

比較演算子チェック

  • 比較演算子は 6種類（==, !=, <, >, <=, >=）
  • 比較の結果は必ず True か False になる
  • = は代入、== は比較（間違えやすい！）
  • 変数同士、文字列同士でも比較できる
