if文を書いてみよう

条件によって処理を分ける、プログラムの「判断力」

プログラムが「考える」とは？

前回学んだ比較演算子を使って、True か False を得ることができました。今度は、その結果を使って処理を分けてみましょう。

条件がTrueの場合だけ処理を実行するのが if文

if文の書き方

if 条件:
    実行したい処理

たった2行でプログラムに「判断力」を与えられます。大切なポイントは3つです。

if文の3つのルール

1. if のあとに条件を書く
2. 条件の最後に :（コロン）を書く
3. 次の行は半角スペース4つで字下げする（インデント）

実際に書いてみよう

uriage = 1200000
mokuhyou = 1000000

if uriage >= mokuhyou:
    print("目標達成！")
目標達成！

売上120万円は目標100万円以上なので、条件はTrue。
だから print("目標達成！") が実行されます。

インデントが命！

インデント（字下げ）は、「if文の中身はここですよ」とPythonに教える合図です。

score = 85

if score >= 80:
    print("合格です")
    print("おめでとう！")

print("判定終了")  # これは常に実行
Pythonではインデントが文法

他の多くのプログラミング言語では波括弧で囲みますが、Pythonはインデントで範囲を示すのが特徴です。
半角スペース4つを使うのが標準です。タブ文字と混ぜるとエラーになります。

インデントエラーを体験しよう

わざとインデントを忘れて、エラーを見てみましょう。

if 10 > 5:
print("大きい")  # インデントなし！
IndentationError: expected an indented block
エラーは怖くない！

IndentationError は「インデントが必要ですよ」と教えてくれています。
半角スペース4つを入れれば解決します。最初は誰でも忘れます。慣れれば自然に書けるようになりますよ。

はじめてのif文

Google Colabで試してみましょう

温度で判断するプログラムを書いてみましょう。

kion = 35

if kion >= 30:
    print("猛暑日です")

kion の値を 25 に変えると、何も表示されません。
条件がFalseのとき、if文の中身はスキップされるからです。

if文の中に複数行書く

if文の中には何行でも書けます。インデントが揃っている行が全て実行されます。

zaiko = 3

if zaiko < 5:
    print("在庫が少なくなっています")
    print(f"残り{zaiko}個です")
    print("発注を検討してください")
在庫が少なくなっています
残り3個です
発注を検討してください
ここまでのポイント

if文の基本チェック

  • if 条件: の形で書く（コロンを忘れずに）
  • 条件がTrueのとき、インデントされた処理が実行される
  • インデントは半角スペース4つが標準
  • インデントを忘れるとIndentationError

業務シナリオでif文を書こう

経費チェックプログラム

経費の金額が1万円を超えていたら、上長承認が必要と表示するプログラムを書いてみましょう。

keihi = 15000

if keihi > 10000:
    print("上長の承認が必要です")
    print(f"申請額: {keihi}円")

keihi を 8000 に変えたら、何が起きるか試してみてください。

まとめ：if文の基本

  • if文 は条件がTrueのとき、中の処理を実行する
  • インデント（半角スペース4つ）でif文の範囲を示す
  • 条件がFalseのとき、if文の中身はスキップされる
  • 次のトピックでは、条件がFalseの場合の処理も書けるようになります
