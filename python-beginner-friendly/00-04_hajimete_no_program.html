はじめてのプログラムを動かそう

print関数を使って、画面にメッセージを表示してみましょう

いよいよプログラミング！

前回までのトピックで、Google Colabの使い方を一通り学びました。コードセルにコードを入力して、Shift + Enter で実行できましたね。

今回は、いよいよ本格的なプログラムを書いてみます。といっても、最初はとてもシンプルです。たった1行で、画面にメッセージを表示するプログラムです。

ワクワクしますね！

これまでの準備が、ここで実を結びます。あなたの最初のプログラムを、一緒に作りましょう。

print関数とは

print() は「画面にメッセージを表示する命令」です。丸括弧の中に表示したい内容を書きます。プログラミングの世界では、こうした命令のことを「関数」と呼びます。

はじめてのプログラム

世界中のプログラマーが最初に書くお約束のプログラムがあります。それがこちらです。

print("Hello!")

実行すると、こう表示されます。

Hello!
知っておこう

文字を表示するときは、ダブルクォーテーション " で文字を囲みます。シングルクォーテーション ' でもOKです。

PRACTICE

やってみよう：Hello!を表示する

動画を止めて、はじめてのプログラムを実行してみましょう。

手順

1. Google Colabでノートブックを開く

2. コードセルに以下を入力：

print("Hello!")

3. Shift + Enter で実行

4. 「Hello!」と表示されたら成功！

おめでとうございます！

「Hello!」が表示されたら、あなたは今、はじめてのプログラムを書いて実行しました。立派なプログラマーの第一歩です。

日本語も表示できる

Pythonでは、日本語もそのまま表示できます。

print("こんにちは、Python！")
こんにちは、Python！

好きなメッセージを自由に表示してみましょう。自分の名前やチーム名を入れてみるのも面白いですよ。

PRACTICE

やってみよう：好きなメッセージを表示

動画を止めて、自分だけのメッセージを表示してみましょう。

チャレンジ

以下のコードの「ここに好きなメッセージ」の部分を、自分の好きな文章に変えて実行してみてください。

print("ここに好きなメッセージ")

例：自分の名前、好きな食べ物、チーム名、など何でもOK！

自由に試してみよう！

何を表示しても大丈夫です。間違いも実験のうち。「これを入れたらどうなるかな？」と好奇心を大切にしてください。

複数行の表示

print() を複数回使うと、複数の行を表示できます。

print("おはようございます")
print("今日もがんばりましょう")
print("Python研修スタート！")
おはようございます
今日もがんばりましょう
Python研修スタート！
ポイント

プログラムは上から順番に1行ずつ実行されます。1行目の print() が実行された後、2行目、3行目と順番に実行されます。

よくある間違いと対処法

エラーが出ても大丈夫。よくある間違いと、その直し方を知っておきましょう。

クォーテーションの閉じ忘れ

print("Hello!) のように閉じの " を忘れるとエラーになります。
対処法：" が開始と終了の両方にあるか確認しましょう。

括弧の閉じ忘れ

print("Hello!" のように閉じの ) を忘れるとエラーになります。
対処法：( と ) がペアになっているか確認しましょう。

エラーは怖くない！

エラーが出たら、入力ミスを探して直せばOK。プロのエンジニアも毎日エラーと向き合っています。

PRACTICE

やってみよう：エラーを体験する

わざとエラーを出してみましょう。エラーの体験は大切な学びです。

チャレンジ

1. まず、わざと間違ったコードを実行してみてください：

print("Hello!)

2. エラーメッセージが表示されることを確認

3. 次に、正しいコードに直して実行してみましょう：

print("Hello!")
すばらしい！

エラーを出して、直して、正しく動かす。この一連の流れが、プログラミングの基本です。
