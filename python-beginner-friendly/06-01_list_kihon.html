データをまとめる

50人分の名前を50個の変数で管理する？
もっとスマートな方法があります

こんな経験ありませんか？

社員50人の名前を管理したい。
変数で1つずつ作ると...

変数を50個も作る？
# これはつらい...
name1 = "田中"
name2 = "鈴木"
name3 = "佐藤"
# ... name50まで続く！？

Excelなら1つの列にまとめて管理できますよね。
Pythonでもそれができます。

リストなら1行で解決！

リストを使えばこうなる
names = ["田中", "鈴木", "佐藤"]
Excelで言うと

Excelの「1つの列」に複数のデータが入っているイメージです。
A1セルに田中、A2セルに鈴木、A3セルに佐藤 ... これが1つの変数にまとまります。

リストのイメージ

リストは「番号付きの棚」のようなもの

リストの作り方

角かっこ [ ] の中にデータをカンマで並べます。

# 文字列のリスト
fruits = ["りんご", "バナナ", "みかん"]

# 数値のリスト
scores = [85, 92, 78, 90]
ポイント

文字列も数値も、どんなデータでもリストにまとめられます。
第5章のfor文で for name in ["田中", "鈴木"] と書いたとき、
実はリストを使っていました！

やってみよう

リストを作ってみよう

チャレンジ：自分のチームメンバーリストを作ろう

Google Colabで以下を入力して実行してみましょう。

team = ["田中", "鈴木", "佐藤"]
print(team)
['田中', '鈴木', '佐藤']
いいですね！

リストを表示すると、角かっこ付きで中身が全部見えます。
自分のチームメンバーの名前に変えて試してみましょう！

len() で要素の数を数える

リストに何個データが入っているか知りたいとき、
len() 関数を使います。

team = ["田中", "鈴木", "佐藤"]
print(len(team))
3
覚えていますか？

第3章で len("こんにちは") で文字数を数えましたね。
同じ len() がリストにも使えるんです！

空のリストから始めることも

最初は空のリストを作って、後からデータを追加することもできます。

# 空のリストを作る
members = []
print(len(members))
0

空のリストには何も入っていないので長さは0です。
次のトピックで要素を追加する方法を詳しく学びます。

リストとfor文の相性はバツグン

第5章で学んだfor文を使えば、
リストの中身を1つずつ順番に処理できます。

team = ["田中", "鈴木", "佐藤"]
for name in team:
    print(f"{name}さん、お疲れ様です！")
田中さん、お疲れ様です！
鈴木さん、お疲れ様です！
佐藤さん、お疲れ様です！
やってみよう

リストとfor文を組み合わせよう

チャレンジ：売上データの合計を求めよう

以下のコードを入力して実行してみましょう。

sales = [120000, 85000, 200000]
total = 0
for s in sales:
    total = total + s
print(f"合計: {total}円")
合計: 405000円

第5章で学んだ合計の計算パターンですね。
リストにデータをまとめておけば、for文で簡単に集計できます。

ここまでのポイント

リストの基本、確認しましょう

  • リストは [ ] で作り、カンマで区切る
  • 1つの変数で複数のデータをまとめて管理できる
  • len() でリストの要素数がわかる
  • for文と組み合わせると全要素を順番に処理できる

まとめ：リストとは

  • リスト = 複数データの入れ物 角かっこ [ ] でデータをまとめる。Excelの1列のようなイメージ。
  • len() で要素数がわかる 文字列のときと同じlen()関数がリストにも使える。
  • for文との相性バツグン リストの全データを1つずつ順番に処理できる。
