数字には2つの種類がある

変数で計算をしてきましたが、実は数字にも種類があります。Pythonでは数字を2つのタイプに分けています。

int
整数（int）
小数点がない数字
10, 0, -5, 1000
個数、人数、年齢など
float
小数（float）
小数点がある数字
3.14, 1.1, 0.5
税率、割合、平均値など

Excelでは区別しませんが、Pythonでは内部的に区別しています。

type()で型を調べよう

type()という関数を使うと、データの種類を調べられます。

print(type(100))
print(type(3.14))
<class 'int'>
<class 'float'>

int は整数、float は小数（浮動小数点数）という意味です。

floatの名前の由来

floatは「浮動小数点数」（floating point number）の略です。小数点が浮いて動く（位置が変わる）数字、という意味です。今は「小数のことをfloatと呼ぶ」と覚えておけば大丈夫です。

割り算は必ずfloatになる

Pythonで注意すべきポイントがあります。割り算（/）の結果はいつもfloatです。

print(10 / 2)
print(type(10 / 2))
5.0
<class 'float'>

10 割る 2 は 5 ですが、Pythonでは 5.0（float）になります。割り切れても .0 が付きます。

今は深く考えなくてOK

「割り算すると.0が付くんだな」と知っておくだけで十分です。後の章で型変換を学ぶと、intに戻す方法も分かります。

整数の割り算と余り

Pythonには特別な割り算の演算子があります。

演算子 意味 例 結果
/ 普通の割り算 7 / 2 3.5
// 整数の割り算（切り捨て） 7 // 2 3
% 余り 7 % 2 1
業務での使いどころ

例えば「17人を4人チームに分けると何チームできて、何人余る？」という計算に使えます。
17 // 4 = 4チーム、17 % 4 = 1人余り

23個のお菓子を5人で均等に分けます。

sweets = 23
people = 5
each = sweets // people

1人4個ずつで、3個余ります。数字を変えて色々試してみてください。

intとfloatが混ざると？

整数と小数を混ぜて計算すると、結果はfloatになります。

price = 1000       # int
tax_rate = 1.1    # float
total = price * tax_rate
print(total)
print(type(total))
1100.0
<class 'float'>

消費税計算ではよくこの現象が起きます。intとfloatが混ざると、Pythonは精度の高いfloatに統一して計算します。

ここまでのまとめ

ここまでのポイント

  • int（整数）と float（小数）の2種類がある
  • type() で型を確認できる
  • 割り算（/）の結果は常にfloat
  • //（切り捨て割り算）と %（余り）が便利

業務例：消費税で小数が出る

税込価格の計算では、必ず小数が出てきます。

price = 298
tax_included = price * 1.1
print(tax_included)
print(type(tax_included))
327.8
<class 'float'>
小数の表示を整えたいとき

「327.8円」ではなく「328円」と表示したい場合もありますよね。整数に変換するint()関数は、後の章で学びます。今は「消費税計算ではfloatが出る」と覚えておきましょう。

色々な割り算を比べよう

動画を止めて、自分で書いてみましょう

3種類の割り算を比べてみましょう。

total_minutes = 135
print(total_minutes / 60)   # 普通の割り算
print(total_minutes // 60)  # 何時間？
print(total_minutes % 60)   # 何分余る？

135分は 2時間15分 ですね。// と % で「時間と分」に分けられました。
