型変換とinput関数

キーボードからデータを受け取り、対話的なプログラムを作ろう

「入力した数字」で計算したい

ここまでのプログラムは、数値をコードに直接書いていました。
でも本当は、ユーザーがキーボードから入力した値で計算したいですよね。

input関数の結果は常に文字列。int()で数値に変換が必要

型変換の3つの関数

第2章で整数と小数を学びましたね。文字列と数値を行き来する3つの関数を覚えましょう。

# 数値 → 文字列
str(100)     # "100"（文字列）

# 文字列 → 整数
int("100")   # 100（整数）

# 文字列 → 小数
float("3.14") # 3.14（小数）
覚え方
str = string（文字列）、int = integer（整数）、float = floating point（浮動小数点）の略です。

input() - キーボードから入力を受け取る

name = input("お名前は？ ")
print(f"こんにちは、{name}さん！")
お名前は？ 田中
こんにちは、田中さん！
対話的なプログラム
input関数を使うと、プログラムが止まってユーザーの入力を待ちます。入力してEnterを押すと、その値が変数に入ります。
やってみよう

input関数を使ってみよう

あいさつプログラムを作ってみましょう
name = input("お名前は？ ")
dept = input("部署は？ ")
print(f"{dept}の{name}さん、ようこそ！")

実行すると入力待ちになります。名前と部署を入力してEnterを押してください。

input()の落とし穴

ここが最大の注意点です。input関数の結果は、常に文字列です。

age = input("年齢は？ ")
print(age + 10)  # エラー！
TypeError: can only concatenate str (not "int") to str
inputは常に文字列を返す
たとえ「25」と数字を入力しても、Pythonはそれを文字列の"25"として受け取ります。計算するにはint()やfloat()で数値に変換する必要があります。

int()で変換して計算する

age_str = input("年齢は？ ")
age = int(age_str)
print(f"10年後は{age + 10}歳ですね")
年齢は？ 25
10年後は35歳ですね
一行でも書ける
age = int(input("年齢は？ ")) のように、inputとintを一行にまとめることもできます。慣れてきたらこの書き方も使ってみてください。

業務例：交通費計算プログラム

distance = int(input("片道の距離(km)？ "))
rate = 15
total = distance * 2 * rate
print(f"往復交通費: {total}円")
片道の距離(km)？ 20
往復交通費: 600円
業務活用ヒント
距離を入力するだけで交通費が自動計算される便利なツール。input + int + f文字列の組み合わせで、実用的なプログラムが作れます。

消費税計算プログラムを作ろう

金額を入力して税込価格を計算しましょう
price = int(input("金額は？ "))
tax = price * 0.10
total = price + tax
print(f"税抜: {price}円")
print(f"税込: {total}円")

色々な金額を入力して試してみてください。

float()で小数を受け取る

小数点がある入力にはfloat()を使います。

height = float(input("身長(m)？ "))
weight = float(input("体重(kg)？ "))
bmi = weight / (height * height)
print(f"BMI: {bmi}")
身長(m)？ 1.70
体重(kg)？ 65
BMI: 22.49134948096886
int vs float
整数だけの入力なら int()、小数点がある入力なら float() を使いましょう。
ここまでのポイント

型変換とinput チェックポイント

  • str()で数値を文字列に、int()で文字列を整数に、float()で小数に変換
  • input()でキーボードからの入力を受け取れる
  • inputの結果は常に文字列。計算するにはint()やfloat()が必要
  • int(input("...")) で入力と変換を一行にまとめられる

まとめ

  • 型変換3つ : str() / int() / float() でデータの型を変換できる
  • input()関数 : キーボードからの入力を受け取る。結果は常に文字列
  • 計算の公式 : int(input("...")) で入力値を数値として受け取り、そのまま計算に使える
