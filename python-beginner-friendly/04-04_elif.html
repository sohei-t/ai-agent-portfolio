elif
3つ以上の選択肢

成績A/B/C/Dのように、多段階の判定を書こう

2択では足りないとき

if-elseは2択でした。でも、3つ以上の条件分岐が必要な場面はたくさんあります。

業務での例

成績を90点以上はA、80点以上はB、70点以上はC、それ未満はDに分類したい。
2択のif-elseを何個も書くのは大変ですよね。

こんなときに使うのが elif（エリフ）です。
else + if を省略した形です。

elif の書き方

if 条件1:
    条件1がTrueのとき
elif 条件2:
    条件2がTrueのとき
elif 条件3:
    条件3がTrueのとき
else:
    どれにも当てはまらないとき
elifのルール

上から順に条件が判定される。最初にTrueになった条件の処理だけが実行される。
elifは何個でも追加できる。elseは最後に1つだけ。

上から順に判定される

最初にTrueになった時点で、残りの条件はチェックされません

成績判定プログラム

score = 75

if score >= 90:
    print("成績: A")
elif score >= 80:
    print("成績: B")
elif score >= 70:
    print("成績: C")
else:
    print("成績: D")
成績: C

score は 75。90以上？ いいえ。80以上？ いいえ。70以上？ はい！
ここでCと表示され、残りのelseはスキップされます。

順序に気をつけよう

elifの順序を間違えると...

条件を広い順（小さい順）に書いてしまうと、意図した結果になりません。

score = 95

# 間違った順序
if score >= 70:
    print("C")  # 95でもここに来てしまう！
elif score >= 80:
    print("B")  # ここには来ない

95は70以上なので、最初のifでTrueになり「C」と表示されてしまいます。
厳しい条件を先に書くのがコツです。

ここまでのポイント

elif チェック

  • elif は3つ以上の条件分岐に使う
  • 上から順に判定され、最初にTrueの処理だけ実行
  • 厳しい条件（大きい数）を先に書く
  • elifは何個でも追加できる
やってみよう

配送料の判定プログラム

注文金額で配送料を変えよう

5000円以上で送料無料、3000円以上で300円、それ未満は500円。

kingaku = 3500

if kingaku >= 5000:
    print("送料無料")
elif kingaku >= 3000:
    print("送料: 300円")
else:
    print("送料: 500円")
