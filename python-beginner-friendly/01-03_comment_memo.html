コメントを書こう

未来の自分へのメモ

1ヶ月後の自分は他人

これまでprintや計算のコードを書いてきました。でも、1ヶ月後にそのコードを見たとき、「これは何の計算だっけ？」と思うことがあります。

そこで役立つのがコメントです。コードにメモを残す機能です。

コメントの書き方

Pythonでは、#（シャープ記号）の後ろに書いた文字はコメントになります。コメントはプログラムの実行には一切影響しません。

# 消費税込みの金額を計算する
print(980 * 1.1)
1078.0
# の後ろは完全に無視される
Pythonは # を見つけると、そこから行末までを完全にスキップします。日本語でも英語でも、何を書いても大丈夫です。

コードの右にもコメントを書ける

行の先頭だけでなく、コードの右側にもコメントを書けます。

print(980 * 1.1)   # 税込価格を表示
print(980 * 0.1)   # 消費税額だけを表示
1078.0 98.0
コメントを書くコツ
コードの右にコメントを書くときは、スペースを2つ以上空けてから # を書くと読みやすくなります。

チャレンジ

前回の計算コードにコメントを追加してみましょう。

# 商品の税込価格を計算するプログラム
print("税抜価格:", 1500)
print("税込価格:", 1500 * 1.1)

コメントを書いても書かなくても、実行結果は同じです。確認してみてください。

良いコメントの書き方

あまり良くないコメント
# 980に1.1を掛ける
print(980 * 1.1)
コードを見れば分かることを、そのまま日本語にしただけのコメントです。
良いコメント
# ランチ代の税込価格を計算（消費税10%）
print(980 * 1.1)
「なぜこの計算をしているか」「何のための数字か」が分かるコメントです。

コメントアウト - 一時的にコードを無効にする

テスト中に一部のコードを一時的に無効にしたいとき、行の先頭に # をつけるとそのコードは実行されなくなります。これをコメントアウトと呼びます。

print("注文一覧")
print("コーヒー:", 400)
# print("ケーキ:", 500)
print("合計:", 400)
注文一覧 コーヒー: 400 合計: 400
業務での活用
デバッグ（問題の原因を探す作業）のとき、コードの一部をコメントアウトして動作を確認するのはプロの開発者もよく使うテクニックです。

チャレンジ

コメントを使って、交通費の計算プログラムを分かりやすく書いてみましょう。

# 今月の交通費を計算する
print("=== 交通費精算 ===")
print("電車代:", 320 * 20)  # 片道320円 x 出勤20日
print("バス代:", 210 * 5)   # 片道210円 x 外出5回

自分の通勤費に数字を変えても面白いですよ。合計も計算してみてください。
