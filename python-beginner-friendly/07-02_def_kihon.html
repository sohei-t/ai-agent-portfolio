関数を作ってみよう
def文

自分だけの「自動販売機」を作る魔法の言葉

前回の復習

前回、関数の仕組みを学びました。今度は自分で関数を定義してみましょう。

前回のポイント
関数は「入力 → 処理 → 出力」の仕組み（自動販売機と同じ）。
print, len, int は Pythonが用意した「組み込み関数」でした。
今回は 自分だけのオリジナル関数 を作ります！

def文の書き方

関数を作るには def というキーワードを使います

def 関数名(): のあとにインデントして処理を書きます

最初の関数を作ろう

あいさつをする関数を作ってみましょう。

def greet():
    print("こんにちは！")

これだけで関数の完成です！次に、この関数を使います。

greet()
こんにちは！
大事なポイント
defで関数を「作る」のと、関数名()で「使う」のは別です。
defだけでは何も起きません。呼び出して初めて動きます！

定義と呼び出しは別もの

レシピと料理の関係
defで関数を作るのは「レシピを書く」こと。呼び出すのは「実際に料理する」こと。
レシピは1回書けば、何度でも料理できます！
やってみよう

関数を作って呼び出そう

動画を止めて、Google Colabで試しましょう

以下のコードを入力して実行してください。

def greet():
    print("こんにちは！")

greet()
greet()
greet()
こんにちは！
こんにちは！
こんにちは！
3回表示されましたか？
1回定義するだけで、何回でも呼び出せます。これが関数の「再利用」です！

インデントが大事！

if文やfor文と同じく、関数の中身はインデント（半角スペース4つ）で書きます

def morning_greeting():
    print("おはようございます")
    print("今日もよろしくお願いします")
よくあるミス
インデントを忘れるとエラーになります。
IndentationError が出たら、defの次の行がスペース4つで始まっているか確認しましょう。

複数の処理をまとめる

関数の中には何行でも処理を書けます。

def show_menu():
    print("=== メニュー ===")
    print("1. 売上確認")
    print("2. 在庫確認")
    print("===============")
=== メニュー ===
1. 売上確認
2. 在庫確認
===============
業務での活用
メニュー表示やレポートのヘッダーなど、「毎回同じ表示をする部分」は関数にまとめると便利です。

関数名のつけ方

良い名前をつけると、コードがグッと読みやすくなります

命名のコツ
「動詞 + 名詞」で名前をつけましょう。show_menu, calc_tax, check_stockなど。
変数名と同じく、英語の小文字とアンダースコアで書きます。

オリジナル関数を作ろう

動画を止めて、自分の関数を作ってみましょう

以下のコードを入力して、自分だけの関数を作りましょう。

def show_header():
    print("========================")
    print("  株式会社サンプル")
    print("  日次売上レポート")
    print("========================")

show_header()
========================
  株式会社サンプル
  日次売上レポート
========================
ここまでのポイント

def文の基本

  • def 関数名(): で関数を定義する
  • 関数の中身はインデント（半角スペース4つ）で書く
  • 定義しただけでは動かない。関数名()で呼び出す
  • 関数名は「何をするか」がわかる名前にする

まとめ

  • def文 関数を作る魔法の言葉。def 関数名(): と書く
  • インデント if文やfor文と同じく、中身はスペース4つ
  • 定義と呼び出し レシピを書く（def）と料理する（呼び出し）は別
