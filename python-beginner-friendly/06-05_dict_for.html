ポイント

for name in scores と書くと、キーが順番に取り出されます。
値も一緒に取り出したい場合は、次のスライドで紹介する .items() を使います。

.items() でキーと値を同時に取得

キーと値を同時にループで使いたいときは .items() を使います。

scores = {"田中": 85, "鈴木": 92, "佐藤": 78}
for name, score in scores.items():
    print(f"{name}: {score}点")
田中: 85点
鈴木: 92点
佐藤: 78点
業務での使い方

社員名簿の全員の情報を表示する、全商品の価格一覧を出力する、
といった場面で大活躍します。

辞書をfor文で回してみよう

チャレンジ：社員の部署一覧を表示
dept = {
    "田中": "営業部",
    "鈴木": "開発部",
    "佐藤": "総務部"
}
for name, d in dept.items():
    print(f"{name}さんは{d}です")
田中さんは営業部です
鈴木さんは開発部です
佐藤さんは総務部です
いい感じです！

辞書の全データをfor文で処理できました。
nameとdの2つの変数に、キーと値が同時に入るのがポイントです。

辞書 + for文 + if文

条件分岐と組み合わせると、さらに実用的になります。

scores = {"田中": 85, "鈴木": 92, "佐藤": 65}
for name, score in scores.items():
    if score >= 80:
        print(f"{name}: {score}点 合格")
    else:
        print(f"{name}: {score}点 再テスト")
田中: 85点 合格
鈴木: 92点 合格
佐藤: 65点 再テスト

辞書にデータを追加する実践パターン

空の辞書から始めて、ループでデータを作ることもできます。

names = ["田中", "鈴木", "佐藤"]
scores = [85, 92, 78]

result = {}
for i in range(len(names)):
    result[names[i]] = scores[i]
print(result)
{'田中': 85, '鈴木': 92, '佐藤': 78}
応用テクニック

2つのリストを組み合わせて辞書を作るパターンです。
Excelの2つの列を組み合わせるイメージですね。

やってみよう

辞書の更新を試そう

チャレンジ：全員に10点ボーナスを加算
scores = {"田中": 85, "鈴木": 92, "佐藤": 78}
for name in scores:
    scores[name] = scores[name] + 10
print(scores)
{'田中': 95, '鈴木': 102, '佐藤': 88}
素晴らしい！

for文で辞書のキーを回しながら、値を更新できました。
一括でデータを修正する場面で大活躍するパターンです。

ここまでのポイント

辞書 + for文 を確認

  • for key in dict: でキーを1つずつ取り出せる
  • .items() でキーと値を同時に取得
  • if文と組み合わせて条件付き処理ができる
  • ループ内で辞書の値を更新できる

まとめ：辞書とfor文

  • .items() が最重要 キーと値を同時にループで取り出す。社員名簿の全員処理などに使う。
  • if文との組み合わせ 「80点以上なら合格」のような条件付き処理がfor文の中で書ける。
  • 辞書の値も更新可能 ループの中で辞書の値を書き換えることで、一括データ修正ができる。
