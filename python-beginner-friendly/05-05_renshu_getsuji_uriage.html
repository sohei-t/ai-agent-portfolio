Chapter 5 - Practice

月次売上データの集計

第5章の総合練習 - ループの力を体感しよう

今回の課題

ミッション

12ヶ月分の売上データから、年間合計、月平均、最高売上月を求める月次レポートを作成してください。

この章で学んだ以下の技術をすべて使います。

  • for文でリストを順に処理
  • range関数で月番号を生成
  • 合計・平均の計算パターン
  • for文 + if文の組み合わせ
やってみよう

Step 1: データを準備しよう

Google Colabで入力してください

動画を止めて、以下のコードを入力してください。まだ実行しなくてOKです。

# 12ヶ月分の売上データ（万円）
sales = [120, 98, 145, 132, 156, 178,
         165, 142, 189, 201, 175, 210]

1月が120万円、2月が98万円、...12月が210万円の売上データです。

やってみよう

Step 2: 月別に売上を表示しよう

コードを追加してください

Step 1のコードの続きに、以下を追加してください。

# 月別の売上を表示
for i in range(12):
    print(f"{i + 1}月: {sales[i]}万円")
ポイント

range(12)は0から11の数を生成します。i + 1で「1月」から始まるようにしています。sales[i]でi番目の売上を取得します。

Step 3: 合計と平均を計算

月別表示の後に、合計と平均を追加しましょう。

# 合計を計算
total = 0
for s in sales:
    total += s
average = total / len(sales)

print(f"年間合計: {total}万円")
print(f"月平均:   {average}万円")

前のトピックで学んだ合計・平均パターンそのままです。

Step 4: 最高売上月を見つける

for文とif文を組み合わせて、最も売上が高い月を見つけます。

# 最高売上月を見つける
max_sale = sales[0]
max_month = 1
for i in range(12):
    if sales[i] > max_sale:
        max_sale = sales[i]
        max_month = i + 1

print(f"最高売上: {max_month}月の{max_sale}万円")
仕組み

最初の月（1月）を仮の最高として、各月の売上と比較。より大きければ更新していきます。

やってみよう

完成コードを実行しよう

全体を1つのセルにまとめて実行

動画を止めて、以下の完成コードを新しいセルに入力して実行してください。

sales = [120, 98, 145, 132, 156, 178,
         165, 142, 189, 201, 175, 210]

# 月別表示
for i in range(12):
    print(f"{i+1}月: {sales[i]}万円")

# 合計と平均
total = 0
for s in sales:
    total += s
average = total / len(sales)

# 最高売上月
max_sale = sales[0]
max_month = 1
for i in range(12):
    if sales[i] > max_sale:
        max_sale = sales[i]
        max_month = i + 1

print(f"--- レポート ---")
print(f"年間合計: {total}万円")
print(f"月平均:   {average}万円")
print(f"最高売上: {max_month}月の{max_sale}万円")

チャレンジ: 目標達成判定を追加

追加課題

月次目標を150万円として、各月が目標達成かどうかを表示するコードを追加してみましょう。

target = 150
for i in range(12):
    if sales[i] >= target:
        print(f"{i+1}月: {sales[i]}万円 達成")
    else:
        print(f"{i+1}月: {sales[i]}万円 未達成")

for文とif文の組み合わせです。動画を止めて挑戦してみてください。
