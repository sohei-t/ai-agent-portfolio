Chapter 00

Google Colabを使ってみよう（後編）

セルの追加・削除、保存方法、テキストセルとコードセルの違いを学びます

前回のおさらい

前編では、Google Colabにアクセスしてノートブックを作成し、セルを実行する方法を学びました。1 + 1 を実行して 2 が表示されましたね。

今回は、もう少し便利な使い方を学んでいきます。セルの追加・削除、保存方法、そしてテキストセルとコードセルの違いです。

セルを追加する

ノートブックには、セルを何個でも追加できます。1つのセルに1つの処理を書くのが基本です。

知っておこう

セル間にカーソルを合わせると「+ コード」「+ テキスト」のボタンが表示されます。または、画面上部のメニューバーからも追加できます。

セルを削除する

不要なセルは削除できます。削除したいセルを選択して、以下の方法で削除します。

  1. セルを選択する
    削除したいセルをクリックして選択します。
  2. ゴミ箱アイコンをクリック
    セルの右上にあるゴミ箱アイコンをクリックするか、「編集」メニューから「セルの削除」を選びます。
注意

削除したセルは元に戻せます。Ctrl + Z（Macは Cmd + Z）で取り消しできるので、間違って消しても安心です。

コードセルとテキストセル

Google Colabには2種類のセルがあります。

💻

コードセル

Pythonのプログラムを書いて実行するセル。前編で使ったのはこちらです。実行するとすぐに結果が表示されます。

📝

テキストセル

メモや説明を書くためのセル。プログラムは実行されません。自分用のメモや、コードの説明を書くのに使います。

ポイント

Excelに例えると、コードセルは「数式を入力するセル」、テキストセルは「コメントを書くセル」のようなものです。

PRACTICE

やってみよう：セルを追加して使う

動画を止めて、セルの追加と削除を練習しましょう。

手順

1. 前回作成したノートブックを開く（または新規作成）

2. 「+ コード」ボタンで新しいコードセルを追加

3. 追加したセルに 100 * 3 と入力して実行

4. 「+ テキスト」ボタンでテキストセルを追加

5. テキストセルに「練習メモ」と入力

いい調子です！

セルの追加ができれば、ノートブックを自分の好きなように構成できます。

ノートブックを保存する

安心ポイント

Google Colabは自動保存されるので、保存し忘れてデータが消える心配はほとんどありません。大事な変更をした後は、念のため手動保存しておくと安心です。

ノートブック名を変更する

新しいノートブックを作ると「Untitled0.ipynb」のような名前がつきます。分かりやすい名前に変更しておきましょう。

ポイント

画面左上のファイル名をクリックすると、名前を編集できます。「Python研修_第0章」のように章番号をつけると、後から見返しやすくなります。

PRACTICE

やってみよう：保存と名前の変更

動画を止めて、保存とノートブック名の変更を練習しましょう。

手順

1. Ctrl + S（Macは Cmd + S）で手動保存

2. 画面左上のファイル名をクリック

3. 「Python研修_練習」に名前を変更

4. Enterキーで確定

着実に進んでいます！

これでGoogle Colabの基本操作は一通り覚えました。もう研修で使っていくには十分です。

セルの使い方のコツ
