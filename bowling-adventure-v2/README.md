# Bowling Adventure V2

障害物コースを通り抜ける新感覚3Dボーリングゲーム

## Live Demo

**[Play Now](https://sohei-t.github.io/ai-agent-portfolio/bowling-adventure-v2/)**

## Game Overview

スマートフォンを傾けてボールを操作し、様々な障害物を避けながらピンを目指す、アドベンチャー要素満載のボーリングゲームです。

### Features

- **傾きセンサー操作**: スマホを傾けて直感的に操作
- **3D物理シミュレーション**: Three.js + Cannon.js による本格的な物理演算
- **10フレーム制**: 従来のボーリングと同様のスコアリング
- **多彩な障害物**: 岩、回転バー、パチンコゾーン、バンパーなど

### Obstacles & Gimmicks

| Type | Description |
|------|-------------|
| Ice Zone | 滑りやすい氷の床 |
| Sand Zone | 減速する砂地 |
| Cliff Path | 落ちたら即ガター |
| Pachinko Zone | 釘で方向がランダムに変化 |
| Rotating Bar | タイミングを見て通過 |
| Bumpers | ピンボールのように弾かれる |
| Pistons | 壁から飛び出すピストン |

### Power-ups

- Speed Booster (orange): 一時的に加速
- Shield (blue): 障害物を1回無効化
- Giant (purple): ボールが巨大化

## How to Play

1. **START**をタップ（傾きセンサーの許可が必要）
2. スマホを**前に傾ける**と加速（トップスピードまで約20秒）
3. **左右に傾ける**と方向転換
4. 障害物を避けながらピンエリアへ
5. 高速でピンに当たるほど多く倒せる

### Controls (Desktop)

- Arrow Up: 加速
- Arrow Down: 減速
- Arrow Left/Right: 方向転換

## Tech Stack

- Three.js (3D Graphics)
- Cannon.js (Physics Engine)
- Vite (Build Tool)
- DeviceOrientation API (Tilt Control)
