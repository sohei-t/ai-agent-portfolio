# Bowling Adventure

3D obstacle course bowling game with tilt controls for mobile devices.

## Features

- **3D Physics-Based Bowling**: Realistic pin physics using Cannon.js
- **Tilt Controls**: Use your device's gyroscope for intuitive ball control
- **Virtual Joystick**: Touch controls for devices without gyroscope
- **10-Frame Game**: Full bowling game with proper scoring
- **Obstacle Course**: Navigate through challenging terrain to reach the pins
- **Strike/Spare Celebrations**: Visual feedback for great throws

## How to Play

1. **Start the Game**: Tap "Start Game" on the title screen
2. **Enable Tilt Controls**: Tap "Enable Tilt Controls" for gyroscope mode (optional)
3. **Throw the Ball**: Tap "投球開始" to begin each throw
4. **Control the Ball**:
   - Tilt your device to steer (gravity-based - ball rolls toward the lower side)
   - Or use the virtual joystick in the corner
5. **Knock Down Pins**: Aim for strikes and spares!

## Technology Stack

- Three.js - 3D rendering
- Cannon.js - Physics simulation
- Vite - Build tool
- DeviceOrientation API - Gyroscope controls

## Controls

- **Gyroscope Mode**: Tilt device to control ball direction
- **Joystick Mode**: Drag the virtual joystick to steer
