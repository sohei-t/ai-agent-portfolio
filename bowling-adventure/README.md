# Bowling Adventure

## Play Now

**[ライブデモ](https://sohei-t.github.io/ai-agent-portfolio/bowling-adventure/)** | **[About](https://sohei-t.github.io/ai-agent-portfolio/bowling-adventure/about.html)**

傾けて転がす！障害物ボーリングゲーム。Three.jsとCannon.js物理エンジンで構築された3Dボウリングゲームです。

## 特徴

- **ジャイロスコープ操作**: スマートフォンを傾けてボールを操作
- **キーボード操作**: PCでは矢印キーで操作
- **10フレームボウリング**: ストライク・スペア対応の本格スコアリング
- **8種類のステータスエフェクト**: REVERSE, SHRINK, GIANT, SLOW, SPEED, SLIP, CHAOS, SHIELD
- **3D物理演算**: リアルなボールとピンの挙動

## 操作方法

| デバイス | 操作 |
|----------|------|
| スマートフォン | 端末を傾けてボールを操作 |
| PC | 矢印キーでボールを操作 |

## 技術スタック

- Three.js - 3Dレンダリング
- Cannon.js - 物理エンジン
- Vite - ビルドツール
