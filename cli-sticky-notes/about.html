// features

生産性を高める機能

⌘

グローバルホットキー

どのアプリを使っていてもフォーカスを切り替えずに付箋を作成。ターミナルはアクティブなまま。

📋

クイックペースト

クリップボードの内容を1つのショートカットで新しい付箋に貼り付け。

🎨

カラーコーディング

6色のテーマで視覚的に分類・整理。一目で内容を把握できます。

📌

常に最前面

重要な付箋をピン留めして、他のウィンドウより常に上に表示。

💾

自動保存

付箋は自動的に保存。位置、サイズ、内容はセッション間で保持されます。

🖥

システムトレイ

メニューバーからクイックアクセス。付箋の作成・表示・非表示を瞬時に。
