# 🚀 Space Shooter Game

## 🎯 概要

クラシックなインベーダーゲームにモダンな要素を加えたHTML5シューティングゲーム。
スマートフォンからデスクトップまで完全対応のレスポンシブデザイン。

## ✨ ゲームの特徴

### コアゲームプレイ
- **レスポンシブデザイン**: 320px〜2560pxまで対応
- **サイバーパンク風ビジュアル**: ネオンカラーと光エフェクト
- **動的難易度**: 敵の数に応じて移動速度が変化
- **ステージ制**: ウェーブをクリアして次のステージへ

### パワーアップシステム
- 🔫 **Rapid Fire**: 連射速度アップ（5秒間）
- 🛡️ **Shield**: 無敵時間（3秒間）
- 🎯 **Multi-Shot**: 3方向同時発射（5秒間）

### 特別要素
- 🛸 **UFO**: 20-40秒間隔でランダム出現、撃墜でボーナスポイント
- 🏛️ **破壊可能なシールド**: 4つの防御壁、段階的に破壊
- 🎉 **ステージクリア演出**: 花火エフェクト付きの祝賀画面

## 🎮 操作方法

### デスクトップ
- **移動**: 矢印キー または WASD
- **攻撃**: スペースキー
- **ポーズ**: P または Escape

### スマートフォン
- **移動**: 画面下部の ◀ ▶ ボタン
- **攻撃**: 画面右下の FIRE ボタン
- **メニュー選択**: 画面タップ

## 🚀 技術スタック

- **JavaScript (Pure)**: フレームワーク不使用
- **HTML5 Canvas**: 2Dグラフィックス描画
- **Web Audio API**: 動的サウンド生成
- **CSS3**: レスポンシブデザイン

## 💻 ローカルでの実行方法

### 前提条件
- ブラウザ（Chrome, Firefox, Safari, Edge推奨）

### 起動
```bash
# ブラウザで直接開く
open src/index.html

# または、HTTPサーバー経由（オプション）
npx http-server .
```
