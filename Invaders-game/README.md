# Space Invaders Game 🚀

A classic Space Invaders-style shooting game built with pure JavaScript and HTML5 Canvas.

## 🎮 Live Demo
- **Play Now**: [https://sohei-t.github.io/ai-agent-portfolio/Invaders-game/](https://sohei-t.github.io/ai-agent-portfolio/Invaders-game/)
- **About Page**: [https://sohei-t.github.io/ai-agent-portfolio/Invaders-game/about.html](https://sohei-t.github.io/ai-agent-portfolio/Invaders-game/about.html)

## ✨ Features
- **Responsive Design** - Works on all screen sizes (320px ~ 2560px)
- **Touch Controls** - Full mobile/tablet support with on-screen controls
- **Power-up System** - Collect power-ups for enhanced abilities:
  - 🔥 Rapid Fire
  - 🛡️ Shield
  - 💫 Multi-Shot
- **UFO Bonus** - Special UFO appears for bonus points
- **Destructible Shields** - Strategic defense elements
- **Progressive Difficulty** - Challenge increases as you advance
- **Sound Effects** - Dynamic audio using Web Audio API

## 🎯 Game Controls

### Desktop
- **Move**: Arrow keys (← →) or A/D
- **Shoot**: Space bar
- **Pause**: P or ESC

### Mobile/Tablet
- **Move**: Touch controls on left/right side of screen
- **Shoot**: Tap fire button or anywhere on screen
- **Pause**: Tap pause button

## 🛠️ Technical Stack
- **Pure JavaScript** - No frameworks, vanilla JS
- **HTML5 Canvas** - Hardware-accelerated graphics
- **Web Audio API** - Dynamic sound generation
- **CSS3** - Responsive styling

## 📊 Performance
- **60 FPS** gameplay
- **Optimized rendering** with efficient collision detection
- **Mobile-optimized** touch handling
- **Test Coverage**: 91.82%

## 🎨 Game Elements
- **Player Ship** - Your spacecraft with 3 lives
- **Enemy Fleet** - 55 invaders in formation
- **UFO** - Appears randomly for bonus points (50-300)
- **Shields** - 4 destructible barriers for protection
- **Power-ups** - Random drops from defeated enemies

## 📱 Browser Compatibility
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
- Mobile browsers (iOS Safari, Chrome Mobile)

## 🏆 Scoring System
- Green Enemy: 10 points
- Blue Enemy: 20 points
- Red Enemy: 30 points
- UFO: 50-300 points (random)
- Power-up Collection: 50 points
