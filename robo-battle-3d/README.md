# ROBO BATTLE 3D

**🎮 ライブデモ: https://sohei-t.github.io/ai-agent-portfolio/robo-battle-3d/**

War Robots にインスパイアされた、ブラウザで遊べる 3D メカ・バトルロイヤル。
都市型アリーナで 4 機の巨大ロボットが入り乱れる Free-For-All。最後の 1 機まで生き残れ。

## 遊び方

### ゲームの流れ
1. **ハンガー(ドック)** で機体クラスを選択(SCOUT / VANGUARD / BASTION)
2. 武器スロットを編集(獲得ポイントで新武器を購入、機体間で載せ替え可)
3. **LAUNCH ⚔** で出撃 — AI 機 3 体とのバトルロイヤル
4. 戦績に応じてポイント獲得(撃破 / 与ダメージ / 生存 / 勝利ボーナス)→ ハンガーで強化 → 再出撃

### 操作(PC)
| キー | 動作 |
|---|---|
| カーソルキー / WASD | 移動(全方位ストレイフ) |
| Shift + ← → | 方向転換 |
| Space / Z | 武器スロット 1 / 2 を発射 |
| Tab | ロックオン対象の切替 |
| B | アビリティ(スプリント / エネルギーシールド) |
| V | ジャンプ |
| マウスドラッグ | 視点微調整(任意) |

### 操作(スマホ・横画面推奨)
左スティック = 移動 / 画面右を横スワイプ = 旋回・照準 / 右下の武器パネルで射撃・アビリティ・ジャンプ

### 戦術のヒント
- 敵が視界に入ると自動ロックオン。**ビルの陰に入ればロックを切れる**(敵からも切られる)
- 円形レーダーで敵の方角を掴み、遮蔽を縫って接近
- 赤い燃料ドラム缶は撃つと誘爆。コンテナは破壊して射線を開ける
- 運河(溝)の中は敵から見えない移動路。橋上のポイントクレートはリスクと引き換え

## 技術構成

- **Three.js**(WebGL)— ブルームポストプロセス、動的ライティング、プロシージャル都市生成
- **AI 生成 3D モデル** — 機体 4 種は Meshy AI で生成(自動リギング + 歩行アニメーション)
- **Web Audio API** — 効果音 16 種とチップチューン BGM をすべてコードで合成(音声ファイル不使用)
- **ローカルセーブ** — 獲得ポイント・購入武器・装備は端末に保存
- 外部サービス・サーバー通信なし。静的ファイルのみで動作
