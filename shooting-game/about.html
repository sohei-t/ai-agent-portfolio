ゲーム概要

「Vertical Scrolling Shooter」は、ブラウザで動作する本格的な縦スクロールシューティングゲームです。 Canvas APIとバニラJavaScriptのみで実装され、60FPSの滑らかな動作を実現しています。

アーキテクチャ概要

本ゲームは、モジュラーアーキテクチャを採用し、以下のレイヤーで構成されています：

Core Layer

ゲームループ、入力処理、衝突判定などの基盤システム

Entity Layer

プレイヤー、敵、弾丸、アイテムなどのゲームオブジェクト

System Layer

スコア管理、レベル管理、敵スポーン、武器システム

Rendering Layer

Canvas描画、パーティクルエフェクト

UI Layer

メニュー、HUD、ゲームオーバー画面

Utility Layer

ベクトル演算、オブジェクトプール、数学関数

パフォーマンス最適化

  • オブジェクトプーリング: 弾丸やエフェクトの再利用でGC負荷を軽減
  • 空間分割: 衝突判定の計算量をO(n²)からO(n)に削減
  • レンダリング最適化: 画面外オブジェクトの描画スキップ
  • requestAnimationFrame: ブラウザのリフレッシュレートに同期
  • イベント委譲: イベントリスナーの効率的な管理

開発プロセス

本プロジェクトは、AIエージェントワークフローによる完全自動開発で実現されました。 人間の手を介さず、計画からテスト、完成まで、すべてAIが自律的に実行しています。

4フェーズ自動開発ワークフロー

Phase 1: 要件定義・計画

要件定義、WBS作成、テスト設計、システム設計を並列実行。各タスクは改善ループ（最大3回）で品質を保証。

Phase 2: 実装

フロントエンド、バックエンド、データベースを並列開発。各コンポーネントは独立したエージェントが担当。

Phase 3: テスト・改善

評価エージェントがテストを実行。不合格の場合、改善プランナーとフィクサーが自動修正（最大3回ループ）。

Phase 4: 完成処理

ドキュメント生成、ランチャー作成、最終レビューを並列実行。全チェック合格後、自動リリース。
